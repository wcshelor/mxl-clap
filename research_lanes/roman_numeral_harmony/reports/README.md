# Reports

This directory is for derived CSV and Markdown outputs from Roman numeral harmony experiments.

Typical outputs include:

- `*_rn_events_summary.csv`
- `*_rn_harmony_features.csv`
- `*_rn_pairwise_similarities.csv`
- `*_feature_audit.csv`
- `*_feature_embedding_correlations.csv`
- `*_rn_harmony_report.md`
