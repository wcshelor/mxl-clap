# data

Put full source MusicXML / MXL pieces in `data/raw/full-pieces/`; the current Mozart/Chopin experiment corpus is documented in `data/raw/full-pieces/README.md`.

Generated excerpts belong in `data/raw/excerpts/generated/`.

Prefix excerpts for the "first 32 / first 64 measures" workflow belong in `data/raw/excerpts/prefixes/`.

Generated artifacts such as CSV tables, manifests, embeddings, MIDI files, and WAV files belong in `data/processed/`.
